Message for Lucy and Nathan (From Lyons on 2/24/2026): Updating this README file to have as easy reference for pulling, staging, commiting, and pushing files we work on. Sorry if it's a little backseaty lol. 

- For this project, I am opting to NOT set up separate branches for simplicity sake.
  * If you guys want to make separate branches, let's definetely talk about that
 
- Before starting anything, be sure to update your local repository. Command to do that: `git pull main`
  * If there are any merge conflicts, please let someone know. Or if you're confident and goated, go ahead and fix it
 
- After making changes/updates, do remember to stage, commit, and push the file. Commands to do that:
  1. `git add filename` (git add -A if you just want to add every file updated)
  2. `git commit -m "Commit message here, ideally informative on what was changed/updated for future reference + who contributed to what changes"`
  3. `git push`
